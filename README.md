# Pattern Library

This is a play-ground for testing out fabricator and creating my own pattern library.

## Quick Start

```shell
$ npm start
```

## Colors
```
//theme code-colors
$theme-colors:{

	secondary: #212121,

	p-light: #5fb14b,
	p-dark: #005300,

	s-light: #484848,
	s-dark: #000
}```
